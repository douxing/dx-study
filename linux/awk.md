## 变量名称

- $0	当前记录（这个变量中存放着整个行的内容）
- $1~$n	当前记录的第n个字段，字段间由FS分隔
- FS	field sep
- RS	record sep
- OFS	out field sep
- ORS	out record sep
- NF	number of field
- NR	number of records
- FNR	file number of records
- FILENAME	当前输入文件的名字
